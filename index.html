Adventurer's Scrapbook

Draw any notes during your adventure on your handy scrapbook!

You can also change the size and colour of your below. make sure you have fully scrolled up on the window before drawing on the scrapbook.

I was able to create this small art canvas with the ability to change your pen colour and size with the help from this video tutorial. I chose this tutorial because it was one of the only p5.js tutorials I could find that had the features I wanted from a basic art program. The opportunities beyond this proof-of-concept include either being able to make good online drawing programs and/or being able to add a good drawing tool to any kind of website.

Welcome to ChatGPT's text-based adventure game! After reading the bot's outcome scenario you will be asked type your next action. If GPT decides to restart or move back just repeat your message until it properly continues the story. Type 'end' at any time in the story to end it there.

I followed a bit from this text tutorial post in order to properly learn how to type my prompt to make a small, randomized text-based adventure game. Theres also a prompt and description commented out in the code for a version where the bot gives you 4 choices too. The rationale behind choosing this tutorial is that I wanted to see how well chatbots could play text-based adventure games (with the results of it even remembering a lot of what happens in the story surprising me) and the tutorial itself being broad and easily editable. This proof-of-concept shows how chatbots could potentially be used to generate outcomes in a story from user decisions on the fly.
